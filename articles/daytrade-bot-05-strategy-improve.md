## 「なぜ負けているのか」を徹底的に調べた

ボットが動き始めてから気づいたことがある。

**バックテストの成績が良くても、本番に近い条件でテストすると全然違う結果が出る。**

今回はそれを調査して、具体的な改善策を見つけた話を書く。

> ⚠️ 投資は自己責任です。この記事は特定の投資を推奨するものではありません。

---

## バックテストの「嘘」を発見した

最初に作ったバックテスト（`backtest_sliding.py`）では1.8年間で**+326,800円**という良い結果が出ていた。

しかしよく見ると条件が甘かった。

```
バックテストの問題点：
1. 銘柄ごとに独立して売買 → 実質無制限ポジション
2. その足の終値でエントリー → 実際には見えてから買えない
3. スリッページ・手数料なし
4. 日次損失上限が銘柄単位に近い
```

本番に近い条件（次の足の始値で約定・最大2ポジション・コスト考慮）でテストし直すと：

```
現行戦略（trend_follow_bold）:
  取引数: 179回
  勝率:  47.5%
  PF:    0.88  ← 1.0未満＝損する
  損益:  -6,349円
```

**PF0.88は負け戦略だった。**

---

## 22パターンを比較して原因を特定

「何が悪いのか」を特定するため、エントリー条件を変えた22パターンをバックテストで比較した。

主な発見：

### ① 500円未満の銘柄が全損

```
<500円帯: 58取引  勝率41.4%  PF0.66  -6,446円
500-999円: 120取引  勝率50.0%  PF0.98  -652円
```

500円未満の銘柄だけで-6,446円の損失。安い株は板が薄く、スプレッドが大きいため不利。

### ② 出来高急増（1.5倍以上）が逆効果

```
出来高<1.5倍: 35取引  勝率60.0%  PF1.38  +3,040円 ← 良い
出来高1.5-1.9倍: 55取引  勝率43.6%  PF0.65  -7,472円 ← 最悪
```

直感に反する結果だが、出来高が急増している時は機関投資家の「出口売り」が重なっている可能性がある。

### ③ RSI55-64の高値圏が弱い

```
RSI 55-64: 60取引  勝率38.3%  PF0.64  -6,798円 ← 最悪
RSI 75以上: 14取引  勝率64.3%  PF1.60  +1,780円 ← 良い
```

### ④ 午前10時が最も弱い

```
10時台: 89取引  勝率44.9%  PF0.81  -6,171円
13時台: 19取引  勝率63.2%  PF2.71  +3,371円
```

## 改善版の結果

これらの発見をもとに最適な組み合わせ（`combined_volume_15_35`）を実装した。

| 条件 | 変更内容 |
|------|---------|
| 出来高 | 1.3倍以上 → **1.5〜3.5倍に絞る** |
| 価格帯 | 制限なし → **500円以上のみ** |
| 時間帯 | 10:00〜 → **11:00〜** |
| 除外銘柄 | なし → **損失上位10銘柄を除外** |

結果：

```
改善後（combined_volume_15_35）:
  取引数: 72回（179→72、厳選）
  勝率:  56.9%
  PF:    2.62  ← 大幅改善
  損益:  +20,026円
  最大DD: 1,948円（元の83%減）
```

**取引数を60%減らして、損益を+26,375円改善した。**

---

## なぜ「出来高急増を避ける」が有効なのか

直感的には「出来高が多い＝注目されている＝動きやすい」と思いがちだ。

しかし実際には：

```
出来高急増のシナリオ：
  何かニュースや材料が出る
  → 素早い参加者が買う
  → 出来高が急増
  → その後「利確売り」が出る
  → 遅れてブレイクアウトを確認してエントリー
  → 売りに巻き込まれて損切り
```

要するに**出来高急増を見てから入ると、乗り遅れた後の高値掴みになりやすい**。

適度な出来高増加（1.5〜3.5倍）の方が「継続的な買い意欲がある」サインとして機能する。

## ハイブリッド戦略 v2 も追加

同時に、相場環境を自動判定する戦略も改善した。

```
旧: SMA傾きのみで判定
新: SMA傾き + ADX(14) で判定

ADX > 25: トレンド相場 → ブレイクアウト/押し目買い
ADX < 20: レンジ相場  → 平均回帰
下落トレンド          → エントリーしない
```

さらに利確ラインも相場別に変更：

```
トレンド: 固定利確なし（トレーリングで伸ばす）
レンジ:  +1.0%で素早く確定（旧+1.5%から縮小）
```

## 今日学んだこと

1. **バックテストの条件が甘いと数字は良く出る** → 本番に近い条件で再検証が必要
2. **出来高急増は必ずしも良いシグナルではない** → 適度な増加（1.5〜3.5倍）が最適
3. **取引数を減らして質を上げる方が有効** → 72回 / PF2.62 > 179回 / PF0.88
4. **「何をやめるか」の発見がパフォーマンス改善に直結した**
